Math.DS: Dynamical Systems

Attacking Classical Problems in Dynamical Systems with Descriptive Set Theory

April 10, 2020 11:00pm UTC
April 10, 2020 4:00pm UTC -07 (Host local time)
Speaker: Matthew Foreman (UC Irvine) - Hosted by UCLA

Abstract:
In his classical 1932 paper, von Neumann asked 3 questions: Can you classify the statistical behavior of differentiable systems? Are there systems where time-forward is not isomorphic to time-backward? Is every abstract statistical system isomorphic to a differentiable system? These questions can be addressed with some surprising consequences by embedding them in Polish Spaces. (...)

The Collatz Conjecture

April 14, 2020 10:00pm UTC
April 14, 2020 3:00pm UTC -07 (Host local time)
Speaker: Terence Tao (UCLA) - Hosted by MATRIX

Abstract:
Define the Collatz map Col on the natural numbers by setting Col(n) to equal 3n+1 when n is odd and n/2 when n is even. The notorious Collatz conjecture asserts that all orbits of this map eventually attain the value 1. This remains open, even if one is willing to work with almost all orbits rather than all orbits.
